# Star Sign Finder

A simple Python programme that calculates a user's zodiac sign based on their birth date

## Features
- User inputs birth month and day
- Returns correct zodiac sign
- Basic input validation

## What I learned
- Conditional statements (if/elif)
- User input handling
- Basic programme structure in Python

## Future improvements
- Turn into a web app using Flask
- Improve date validation
- Add descriptions/predictions for each sign
